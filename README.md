# 🛒 SimpleCart

**SimpleCart** is a modern and lightweight e-commerce web application designed to provide a smooth and intuitive shopping experience. It allows users to browse products, add items to their cart, and manage purchases seamlessly.

Built with a focus on clean UI, performance, and scalability, SimpleCart demonstrates core e-commerce functionalities using modern web technologies.
---

## 🚀 Features

- 🛍️ Browse products with clean UI  
- ➕ Add to cart functionality  
- 🧮 Dynamic cart updates  
- 🔍 Search and filter products   
- 🔐 User authentication  
- 💳 Checkout system   
- 📱 Fully responsive design  

---
## 🧑‍💻 Tech Stack

### Frontend
- HTML5  
- CSS3  
- JavaScript (ES6+)  
- React.js

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose ODM)

🔹 Backend is maintained in a separate repository for better scalability and modularity.

### 👉 Backend Repository:
https://github.com/ketanxyz/SimpleCart-Backend

## ⚙️ Running the Project Locally
### 1️⃣ Clone the Repositories
```bash
# Frontend
git clone https://github.com/ketanxyz/Ecommerce-project.git
# Backend
git clone https://github.com/ketanxyz/SimpleCart-Backend.git
```

### 2️⃣. Setup Backend
```bash
cd simplecart-backend

npm install

npm run dev
```
### 3️⃣. Setup Frontend
```bash
cd Ecommerce-project

npm install

npm run dev
```
