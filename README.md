# 🛒 SimpleCart

**SimpleCart** is a modern and lightweight e-commerce web application designed to provide a smooth and intuitive shopping experience. It allows users to browse products, add items to their cart, and manage purchases seamlessly.

Built with a focus on clean UI, performance, and scalability, SimpleCart demonstrates core e-commerce functionalities using modern web technologies.
---

## 🚀 Features

- 🛍️ Browse products with clean UI  
- ➕ Add to cart functionality  
- 🧮 Dynamic cart updates  
- 🔍 Search and filter products   
- 🔐 User authentication  
- 💳 Checkout system   
- 📱 Fully responsive design
